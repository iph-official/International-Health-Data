# International-Health-Data
26-1 국제보건 데이터 관리 및 분석실습 프로젝트
여러 국제·국내 보건, 개발, 기후, 사회경제 데이터셋을 한 곳에서 탐색하고,  
카테고리별로 선택하여 다운로드할 수 있는 **통합 데이터셋 플랫폼** 구축 프로젝트이다.  
초기 목표는 **DHS, World Bank, OECD, OECD CRS** 등을 중심으로 데이터 구조를 정리하고,  
향후 **Streamlit 기반 대시보드**를 통해 변수 검색, 데이터 트리 구조 확인, 다운로드 링크 제공 기능까지 구현하는 것이다.

---

## Project Goal

본 프로젝트의 목표는 서로 다른 기관에서 제공하는 데이터셋을 통합적으로 정리하여,  
사용자가 다음 기능을 쉽게 수행할 수 있도록 하는 것이다.

- 데이터 출처별 구조 확인
- 카테고리별 데이터 탐색
- 변수명 및 변수 설명 검색
- 데이터셋 다운로드 링크 확인
- 향후 일부 데이터의 통합 다운로드 지원

특히 **보건행정, 국제개발협력, 환경보건, 지구건강 연구**에 활용 가능한 데이터 자원을 체계화하는 것을 목적으로 한다.

## Target Data Sources

### International Data

- **DHS (Demographic and Health Surveys)**
- **World Bank DataBank**
- **OECD Data Explorer**
- **OECD CRS (Creditor Reporting System)**
- **Global Burden of Disease (GBD)**
- **Copernicus Climate Data Store**
- **Google Earth Engine Datasets**
- **기상청 데이터** *(추후 검토 예정)*

### Domestic Data

- **지역사회건강조사**
- **한국의료패널**
- **국민건강영양조사**
- **고령화패널**
- 기타 국내 공공데이터

---

## Main Idea

### 1. 데이터 통합 구조화
각 데이터셋의 제공 기관, 접근 경로, 데이터 구조, 주요 변수, 다운로드 방식 등을 정리한다.

### 2. Streamlit 기반 대시보드 구축
사용자가 웹 환경에서 아래 기능을 사용할 수 있도록 구현한다.

- 데이터셋별 트리 구조 확인
- 다운로드 링크 또는 다운로드 방법 확인
- 변수 검색
- 변수 설명 및 범주 확인
- 데이터 출처별 비교 탐색

### 3. 카테고리 기반 탐색
보건, 인구, 경제, 환경, 개발협력 등 카테고리를 설정하여  
관련 데이터셋과 변수를 묶어서 탐색할 수 있도록 한다.
